===========================================
   HOW TO RUN VARARI AI CHAT SERVER
===========================================

OPTION 1: Double-click "run-server.bat"
   - This is the easiest method
   - The batch file will handle everything automatically
   - A window will open showing server status

OPTION 2: Manual Command Line
   1. Open Command Prompt (cmd) or PowerShell
   2. Navigate to the varari-ai-chat folder:
      cd "D:\qwen 235\varari-ai-chat"
   3. Run the server:
      node server.js

OPTION 3: Using npm
   1. Open Command Prompt (cmd) or PowerShell
   2. Navigate to the varari-ai-chat folder:
      cd "D:\qwen 235\varari-ai-chat"
   3. Run:
      npm start

===========================================
   ACCESSING THE APPLICATION
===========================================

Once the server is running:
1. Open your web browser
2. Go to: http://localhost:3000
3. Start chatting with Lara!

===========================================
   STOPPING THE SERVER
===========================================

- Press Ctrl+C in the command window
- Or simply close the command window

===========================================
   TROUBLESHOOTING
===========================================

If the batch file closes immediately:
1. Try Option 2 (Manual Command Line) instead
2. Check if you're in the correct folder
3. Make sure Node.js is installed

The server is confirmed working - you should see:
"Server is running on http://localhost:3000"
